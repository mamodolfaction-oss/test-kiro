Pengendalian Statistik: Sampling Penerimaan (Acceptance Sampling)

Pendahuluan Sampling Penerimaan

Sampling Penerimaan (Acceptance Sampling) adalah metode pengendalian kualitas statistik yang digunakan untuk memutuskan apakah suatu lot (kumpulan produk) diterima atau ditolak berdasarkan pemeriksaan sampel yang diambil secara acak dari lot tersebut.

Mengapa Sampling?

Inspeksi 100% seringkali tidak praktis karena mahal, memakan waktu, atau bersifat destruktif (merusak produk).

Kapan Digunakan?

Saat biaya inspeksi tinggi, lot berukuran besar, atau pengujian bersifat destruktif (misal: uji tarik, uji ledak).

Parameter Utama

N = Ukuran lot, n = Ukuran sampel, c = Bilangan penerimaan (acceptance number).

Risiko

α = Risiko produsen (menolak lot baik), β = Risiko konsumen (menerima lot buruk).

Single Sampling Plan

Pada Single Sampling Plan, keputusan menerima atau menolak lot dibuat berdasarkan satu sampel tunggal. Ini adalah metode paling sederhana.

Parameter

Plan: (N, n, c)
N = ukuran lot, n = ukuran sampel, c = acceptance number

Keputusan:
Jika d ≤ c → TERIMA lot
Jika d > c → TOLAK lot

Rumus Probabilitas Penerimaan (Distribusi Binomial)

P(accept) = ∑d=0c C(n,d) · pd · (1-p)n-d
di mana C(n,d) = n! / (d! · (n-d)!)

Untuk lot kecil (Distribusi Hipergeometrik):
P(accept) = ∑d=0c [C(D,d) · C(N-D, n-d)] / C(N,n)
D = jumlah cacat dalam lot, N = ukuran lot

Aproksimasi Poisson (jika n besar, p kecil):
P(accept) ≈ ∑d=0c e-np · (np)d / d!

Double Sampling Plan

Pada Double Sampling Plan, jika hasil sampel pertama tidak memberikan keputusan yang jelas, diambil sampel kedua sebelum keputusan akhir dibuat.

Parameter

Plan: (n1, c1, r1, n2, c2, r2)
n1 = ukuran sampel pertama, c1 = acceptance number sampel 1
r1 = rejection number sampel 1 (biasanya c1 + 2 atau lebih)
n2 = ukuran sampel kedua, c2 = acceptance number kumulatif

Keputusan Sampel 1:
Jika d1 ≤ c1 → TERIMA lot
Jika d1 ≥ r1 → TOLAK lot
Jika c1 < d1 < r1 → Ambil sampel kedua

Keputusan Sampel 2:
Jika d1 + d2 ≤ c2 → TERIMA lot
Jika d1 + d2 > c2 → TOLAK lot

Rumus Probabilitas Penerimaan

P(accept) = P(accept pada sampel 1) + P(accept pada sampel 2)

P(accept sampel 1) = ∑d=0c1 b(d; n1, p)

P(accept sampel 2) = ∑d1=c1+1r1-1 b(d1; n1, p) · ∑d2=0c2-d1 b(d2; n2, p)

b(d; n, p) = C(n,d) · pd · (1-p)n-d (distribusi binomial)

Multiple Sampling Plan

Multiple Sampling Plan merupakan perluasan dari double sampling dimana sampel diambil secara berurutan (biasanya 3-7 tahap) dengan keputusan terima/tolak/lanjut pada setiap tahap.

Parameter

Plan: (n1, c1, r1), (n2, c2, r2), ..., (nk, ck, rk)
k = jumlah tahap sampling

Pada setiap tahap i:
Hitung Di = ∑ dj (j=1 sampai i) = jumlah cacat kumulatif
Jika Di ≤ ci → TERIMA
Jika Di ≥ ri → TOLAK
Jika ci < Di < ri → Lanjut ke tahap i+1

Rumus Probabilitas Penerimaan

P(accept) = ∑i=1k P(accept pada tahap i)

P(accept tahap i) = P(ci-1 < Di-1 < ri-1) · P(Di ≤ ci | Di-1)

ASN = ∑i=1k ni · P(sampai tahap i)

Keuntungan Multiple Sampling

Rata-rata Sampel Lebih Kecil

ASN biasanya lebih kecil dibandingkan single sampling untuk lot dengan kualitas sangat baik atau sangat buruk.

Psikologis

Lot yang ditolak sudah mendapat "kesempatan kedua" sehingga lebih diterima oleh pemasok.

Kompleksitas Lebih Tinggi

Administrasi lebih rumit dan memerlukan pelatihan personel inspeksi yang lebih intensif.

Kurva OC (Operating Characteristic)

Kurva OC menunjukkan hubungan antara proporsi cacat dalam lot (p) dengan probabilitas penerimaan lot P(accept). Kurva ini adalah alat utama untuk mengevaluasi kinerja suatu sampling plan.

Rumus Matematis

Pa(p) = ∑d=0c C(n,d) · pd · (1-p)n-d
Distribusi Binomial - untuk sampel besar relatif terhadap lot

Pa(p) ≈ ∑d=0c e-np · (np)d / d!
Aproksimasi Poisson - untuk n ≥ 20, p ≤ 0.05

Titik-titik penting pada Kurva OC:
Pa(AQL) = 1 - α ≈ 0.95 (Producer's risk point)
Pa(LTPD) = β ≈ 0.10 (Consumer's risk point)

Ideal OC Curve: Pa = 1 jika p ≤ AQL, Pa = 0 jika p > AQL

Karakteristik Kurva OC

Efek n (Sample Size)

Semakin besar n, kurva semakin curam (mendekati ideal). Discriminating power meningkat.

Efek c (Acceptance Number)

Semakin besar c, kurva bergeser ke kanan (lebih longgar/lenient).

Tipe A vs Tipe B

Tipe A: lot terisolasi (hipergeometrik). Tipe B: proses kontinu (binomial/Poisson).

Kurva OC Interaktif

Bandingkan dua plan: Plan 1 (n,c) vs Plan 2 (n₂,c₂)

AOQ (Average Outgoing Quality) & AOQL (Average Outgoing Quality Limit)

AOQ menggambarkan rata-rata kualitas outgoing setelah inspeksi, dengan asumsi lot yang ditolak di-inspeksi 100% (rectifying inspection). AOQL adalah nilai maksimum dari kurva AOQ.

Rumus Matematis

AOQ(p) = p · Pa(p) · (N - n) / N
Untuk lot yang diterima: item cacat lolos tanpa diganti
Untuk lot yang ditolak: semua item cacat diganti (100% inspeksi)

Aproksimasi (jika n << N):
AOQ(p) ≈ p · Pa(p)

AOQL = max{AOQ(p)} untuk semua 0 ≤ p ≤ 1
AOQL adalah titik terburuk (worst case) kualitas outgoing

Untuk menemukan AOQL, selesaikan:
d/dp [AOQ(p)] = 0
yaitu: Pa(p) + p · dPa/dp = 0 (secara numerik)

Interpretasi

AOQ Rendah saat p kecil

Jika kualitas incoming baik, kualitas outgoing juga baik karena sedikit cacat yang lolos.

AOQ Rendah saat p besar

Jika kualitas incoming buruk, banyak lot ditolak dan di-inspeksi 100%, sehingga cacat dihilangkan.

AOQL = Jaminan Kualitas

Tidak peduli seberapa buruk incoming quality, outgoing quality tidak akan pernah melebihi AOQL.

ATI (Average Total Inspection) & ASN (Average Sample Number)

ATI - Average Total Inspection

ATI mengukur rata-rata jumlah total item yang di-inspeksi per lot, termasuk inspeksi 100% untuk lot yang ditolak (rectifying inspection).

ATI(p) = n · Pa(p) + N · (1 - Pa(p))
= n + (N - n) · (1 - Pa(p))

Interpretasi:
Jika p = 0: ATI = n (hanya sampel yang diperiksa)
Jika p = 1: ATI = N (semua lot diperiksa 100%)
Semakin buruk kualitas → semakin banyak inspeksi

ASN - Average Sample Number

ASN mengukur rata-rata jumlah sampel yang perlu diambil sebelum keputusan dapat dibuat (relevan untuk double dan multiple sampling).

Single Sampling: ASN = n (konstan)

Double Sampling:
ASN(p) = n1 + n2 · P(sampai sampel 2)
ASN(p) = n1 + n2 · [1 - P(accept|sampel 1) - P(reject|sampel 1)]

Multiple Sampling:
ASN(p) = ∑i=1k ni · P(keputusan pada tahap i atau setelahnya)
= ∑i=1k ni · P(belum ada keputusan sebelum tahap i)
