Pengendalian Statistik: Sampling Penerimaan

Pendahuluan Sampling Penerimaan

Apa itu Acceptance Sampling?

Acceptance Sampling (Sampling Penerimaan) adalah teknik pengendalian kualitas statistik yang digunakan untuk menentukan apakah suatu lot/batch produk diterima atau ditolak berdasarkan pemeriksaan sampel acak dari lot tersebut.

Kapan digunakan?
- Pengujian bersifat destruktif
- Biaya inspeksi 100% terlalu mahal
- Lot terlalu besar untuk diperiksa seluruhnya
- Informasi mengenai kualitas proses produksi terbatas

Jenis-jenis Sampling Plan

Single Sampling Plan
Keputusan dibuat berdasarkan satu kali pengambilan sampel. Paling sederhana dan mudah diimplementasikan.
Double Sampling Plan
Mengizinkan pengambilan sampel kedua jika sampel pertama tidak konklusif. Lebih ekonomis.
Multiple Sampling Plan
Memerlukan beberapa tahap pengambilan sampel sebelum keputusan akhir. ASN paling kecil.

Single Sampling Plan

Definisi dan Prosedur

Dalam Single Sampling Plan, diambil satu sampel berukuran n dari lot berukuran N. Jika jumlah cacat d ≤ c, maka lot diterima. Jika d > c, lot ditolak.

Rumus Probabilitas Penerimaan

Distribusi Binomial:

$$P(accept) = \sum_{d=0}^{c} \binom{n}{d} p^d (1-p)^{n-d}$$

Distribusi Hypergeometric (lot terbatas):

$$P(accept) = \sum_{d=0}^{c} \frac{\binom{D}{d}\binom{N-D}{n-d}}{\binom{N}{n}}$$

Aproksimasi Poisson (n besar, p kecil):

$$P(accept) = \sum_{d=0}^{c} \frac{e^{-np}(np)^d}{d!}$$

Double Sampling Plan

Definisi dan Prosedur

Double Sampling Plan memberikan kesempatan kedua sebelum memutuskan. Parameter: n1, c1, r1 (tahap 1) dan n2, c2 (tahap 2).

Probabilitas Penerimaan

$$P(accept) = P(d_1 \leq c_1) + \sum_{d_1=c_1+1}^{r_1-1} P(D_1=d_1) \cdot P(d_1+d_2 \leq c_2)$$

Aturan Keputusan:
- Tahap 1: Jika d1 ≤ c1 maka TERIMA
- Tahap 1: Jika d1 ≥ r1 maka TOLAK
- Tahap 1: Jika c1 < d1 < r1 maka ambil sampel kedua
- Tahap 2: Jika d1 + d2 ≤ c2 maka TERIMA
- Tahap 2: Jika d1 + d2 > c2 maka TOLAK

Tiga Kemungkinan Keputusan pada Tahap 1

KondisiKeputusanPenjelasan
d1 ≤ c1Langsung TERIMAJumlah cacat sangat sedikit, lot jelas berkualitas baik
d1 ≥ r1Langsung TOLAKJumlah cacat terlalu banyak, lot jelas berkualitas buruk
c1 < d1 < r1Ambil sampel keduaHasil belum konklusif, perlu informasi tambahan dari sampel kedua
Contoh: Jika c1=1 dan r1=4, maka:
- d1 = 0 atau 1: langsung terima
- d1 = 4 atau lebih: langsung tolak
- d1 = 2 atau 3: belum pasti, lanjut ke sampel kedua

Multiple Sampling Plan

Definisi dan Prosedur

Multiple Sampling Plan adalah generalisasi dari double sampling, dengan hingga k tahap pengambilan sampel. Setiap tahap memiliki parameter ni, ci, dan ri.

Probabilitas Penerimaan

$$P_a = \sum_{j=1}^{k} P(\text{accept pada tahap } j)$$

Keuntungan: ASN lebih kecil, keputusan lebih cepat untuk lot sangat baik/buruk.
Kerugian: Administrasi lebih kompleks, pelatihan inspektor lebih sulit.

Kurva OC (Operating Characteristic)

Definisi

Kurva OC menggambarkan hubungan antara proporsi cacat dalam lot (p) dengan probabilitas penerimaan lot P(accept). Kurva ini adalah alat utama untuk mengevaluasi kinerja suatu sampling plan.

Binomial (exact): $$P_a(p) = \sum_{x=0}^{c} \binom{n}{x} p^x (1-p)^{n-x}$$

Poisson (aproksimasi): $$P_a(p) \approx \sum_{x=0}^{c} \frac{e^{-np}(np)^x}{x!}$$

AOQ dan AOQL

Definisi

AOQ (Average Outgoing Quality) adalah rata-rata kualitas outgoing setelah inspeksi, dengan asumsi lot yang ditolak diinspeksi 100% dan item cacat diganti.

AOQL (Average Outgoing Quality Limit) adalah nilai maksimum kurva AOQ, yaitu kualitas terburuk yang mungkin lolos.

$$AOQ = \frac{P_a \cdot p \cdot (N-n)}{N}$$

$$AOQL = \max_{0 \leq p \leq 1} [AOQ(p)]$$

ATI dan ASN

Definisi

ATI (Average Total Inspection) adalah rata-rata jumlah total item yang diinspeksi per lot, termasuk inspeksi 100% untuk lot yang ditolak.

ASN (Average Sample Number) adalah rata-rata jumlah sampel yang perlu diperiksa sebelum keputusan dibuat.

ATI: $$ATI = n + (1 - P_a)(N - n)$$

ASN Single: $$ASN = n \quad \text{(konstan)}$$

ASN Double: $$ASN = n_1 + n_2[1 - P(d_1 \leq c_1) - P(d_1 \geq r_1)]$$
